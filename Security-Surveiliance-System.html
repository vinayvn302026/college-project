HD & IP Camera Installation

  • Planning:-
    • Visit the site properly, And assume the areas which you want to cover. and plan location for camera to mount. And ensure cameras have proper connectivity (wired or wireless) and power also.

  • Choose right camera:-
    • Choose the rignt camera according to your needs and budget(HD or IP).

  • Prepare resources:-
    • Collect and arrange proper resources accoarding to your requirement like wire, camera, DVR or NVR, HDD, Cam-Box, Rack etc.

  • Wiring:-
    • Lay camera wires in structured and safe way and terminate one side of the wires where you plan to install rack and other side of the end of camera.

  • Mount Camera and Rack:-
    • Properly mount all the cameras and rack with the help of Hammer machine, screws, Cam-Box etc.

  • Connectors:-
    • Link connectors to wires both ends accoarding to wire, If you have Cat6 wire, then crimp RJ-45 connector with the help of Crimping tool, If you have 3+1 wire, then use DC & BNC connectors.

  • Install Supply & DVR NVR:-
    • Install DVR NVR Power Supply in rack to take input of cameras. If you have HD cameras then you have to install DVR Supply. If you have IP cameras then you have to install NVR, PoE Swith (make sure you have installed HDD in DVR and NVR).

  • Connection:-
    • Then connect connectors to the cameras, DVR NVR, Supply Switch(PoE).

  • Double-Check Connections:-
    • Ensure all connections are securely connected and there are no loose connectors.

  • Power on Equipments:-
    • Power on the equipments and check if everything is working correctly.

  • Configuration:-
    • Configure properly NVR DVR, Cameras(if you have installed IP cameras) and set the prover viewing angle of cameras.

    NVR & DVR Configuration

    1.DVR Configuration

  • Power on DVR.

  • Connect Screen and mouse.

  • Complete initial configuration.

  • Sttup Pattern Lock.

  • Setup username and password.

  • Set date and time.

  • Set IP address of DVR.

  • Format Surveillance HDD.

  • Set recoarding settings.

  • Finally Restart the DVR.

    • 2.NVR configuration

  • Power on NVR.

  • Connect Screen and mouse.

  • Complete initial configuration.

  • Sttup Pattern Lock.

  • Setup username and password.

  • Set date and time.

  • Set IP address of NVR & Cameras.

  • Ensure IP addresses in same network.

  • Format Surveillance HDD.

  • Set recoarding settings.

  • Finally Restart the NVR.

    • VDP Installation

  • Unboxing:-
    • Unbox the VDP and collect all the resources.

  • Plan:-
    • Plan, where you have to install the VDP machine and EM Lock.

  • Mount:-
    • Mount VDP Screen and Camera of VDP in the outside of door.

  • Installation:-
    • Install the EM Lock to the door, where you want to control that door.

  • Internet Cable:-
    • Lay internet cable to provide internet to VDP Screen to connect private network to public network.

  • Connection:-
    • Connect VDP Screen to outside camera with the help of cat6 cable. And connect EM Lock with VDP camera.

  • Rechecking:-
    • Ensure all the connections are connected perfectly and all the connectors also connected tightly.

  • Configuration:-
    • Setup initial configuration of VDP machine and setup basic configuration like date and time, IP address, Username and Password, Pattern, etc.

  • Online:-
    • Online VDP machine to connect VDP machine with mobile application, and finally test the VDP and Lock working.

    Access Control Machine Installation

  • Unboxing:-
    • Unbox the Access Control Machine and collect all the resources.

  • Plan:-
    • Plan, where you have to install the Access Control Machine and EM Lock.

  • Installation:-
    • Install the EM Lock to the door, and Access Control Machine nearby to EM-Lock.

  • Connection:-
    • Connect Access Control Machine to EM-Lock with the help of cat6 cable.

  • Rechecking:-
    • Ensure all the connections are connected perfectly and all the connectors also connected tightly.

  • Configuration:-
    • Setup initial configuration of Access Control Machine and setup basic configuration and passwords with the help of manual codes.

  • Checking:-
    • Ragister the RF-ID cards with the help of manual codes, if you have. And finally test the working of EM-Lock and Access Control Machine.
