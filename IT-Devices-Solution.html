Printer Installation Steps
  • Unbox the printer
  • Connect Printer to System via USB, Ethernet, Wi-Fi.
  • Download and install the printer driver from Printer manufacturer website.
  • Connect printer to Wi-Fi if applicable.
  • Print test page to check the connectivity of system and printer.
    • Printer Troubleshooting
  • Check power connection, and error messages.
  • Ensure it's set as the default printer.
  • Clear the print queue.
  • Update or reinstall drivers.
  • Check paper, ink, and clear jams.
  • Restart the printer and computer.
  • Run the printer troubleshooter (Windows).
  • Verify network connections for wireless printers.
    • Scanner Installation
  • Unpack the scanner and connect it to power.
  • Connect the scanner via USB, Ethernet.
  • Download and install the printer driver from Printer manufacturer website.
  • Open the scanning tool of that manufacturer brand.
  • Test the scanning functulity to check the connectivity between system and scanner.
    • Keyboard Mouse & Other Peripheral Solution
  • Ensure devices are powered and connected properly.
  • Update or reinstall device drivers via Device Manager.
  • Try different USB ports for wired devices.
  • Restart the devices and computer.
  • Run Windows Hardware and Devices troubleshooter.
  • Disabled USB power-saving settings in Device Manager.
  • Update the operating system for compatibility fixes.
  • Test the peripheral on another computer.
    • Wireless Keyboard Mouse Troubleshooting

  • Check Battery:-
    • Replace or recharge batteries in the wireless keyboard and mouse.

  • Verify USB Receiver Connection:-
    • Ensure the USB receiver is securely plugged into the computer.

  • Check Range:-
    • Make sure the devices are within the recommended distance from the receiver.

  • Re-pair Devices:-
    • Re-sync the keyboard/mouse with the receiver using the pairing button.

  • Test USB Port:-
    • Try a different USB port for the receiver.

  • Restart Devices:-
    • Turn the keyboard/mouse off and back on, or remove and reinsert the receiver.

  • Check for Interference:-
    • Remove potential sources of wireless interference (e.g., other wireless devices or metal objects).

  • Update Drivers:-
    • Update of re-install the keyboard/mouse drivers via Device Manager.

  • Test on Another Computer:-
    • Try the keyboard and mouse on another device to rule out hardware issues.

  • Check Bluetooth Settings (if applicable):-
    • Ensure Bluetooth is enabled and working properly for Bluetooth Devices.
